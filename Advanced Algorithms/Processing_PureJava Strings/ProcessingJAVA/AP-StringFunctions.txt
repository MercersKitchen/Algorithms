
AP Subset Strings
String.length() returns number of characters
String.substring( int ) returns the characters after the int counting starts with human numbers (or on with comptuer numbers)
String.substring( int, int ) returns the inbetween characters,
  inclusive with first int and exclusive with second int,
  similar to java.lang.math.random ( int, int )
String.indexOf( “[text]” ) returns index value of text starting position, or -1 if absent
String.toLowerCase() returns string in lower case letters
String.toUpperCase() returns string in Upper Case letters
String.equals( key ) where key is a testing String returns Boolean
String.compareTo ( “[text]” ) returns a value (this should be explored)
String.trim() returns string without beginning and ending, deleting spaces
